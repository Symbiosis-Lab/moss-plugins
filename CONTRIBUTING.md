# Contributing

This repository is a **read-only mirror** of source code maintained in the private moss monorepo. Force-pushes happen on each release.

## How to contribute

- **Bug reports**: file an issue here — issues are triaged and forwarded to the monorepo.
- **Pull requests**: cannot be merged here (the next release sync would overwrite them). Open an issue describing the change; we'll land it in the monorepo for the next release.
- **Security**: see [SECURITY.md](SECURITY.md) — do **not** file public issues for vulnerabilities.
